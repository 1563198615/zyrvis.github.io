# zyrvis.com

蛇 logo 首页 + 8 个苹果风产品页。

## 部署

把根目录的 10 个文件推到 GitHub Pages 仓库根目录:

```
index.html              ← 首页(蛇 logo + 左上角产品菜单)
apple-base.css          ← 苹果风共享 CSS
dongchi.html
coffeechat.html
fitmap.html
penpot.html
vitatrack.html
wavedio.html
football-skill.html
agentteam.html
```

## 路径约定

- 全部在仓库**根目录**(没有子目录)
- 首页 → `index.html`
- 产品页 → `dongchi.html` 等(同目录,直接文件名)
- CSS → `apple-base.css`(同目录)

部署完成后访问 `https://zyrvis.com`(或 `https://www.zyrvis.com`)。
