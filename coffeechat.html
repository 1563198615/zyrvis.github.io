CoffeeChat

LBS 即时社交平台,通过话题锚定帮用户发现附近咖啡馆内的线下社交邀约,实现"半小时内线下履约"。

iOS · Swift 微信小程序 LBS 实时 话题锚定
📄 查看 Markdown 介绍 📂 完整 README
FEATURES

五大核心特性

线上聊一百次,不如线下喝一杯。打开 → 看到附近有人在咖啡馆发起话题 → 步行 5 分钟 → 当面聊 30 分钟。

🗺️
地图发现
基于高德地图,实时展示附近咖啡馆和活跃邀约。看得见的社交。
💬
话题锚定
职场交流 / 创意头脑风暴 / 联合副业探索 / 语言交换 — 用话题筛人。
⚡
即时邀约
步行可达的时间内发起,30 分钟履约。见面成本 = 一杯咖啡的时间。
⭐
信用体系
实名认证 + 履约积分 + 历史评价。失约成本 = 信用分暴跌。
🤝
O2O 联动
商家联运返佣,激活平峰时段客流。咖啡馆老板笑醒。
🚫
不是 Tinder / Soul
不做滑动、不做树洞,只做"附近 + 话题 + 立即"。把线下还回线下。
ARCHITECTURE

技术架构

双端 + Node.js 网关 + Redis/PostgreSQL 经典组合,Socket.io 撑起实时位置与会话。
