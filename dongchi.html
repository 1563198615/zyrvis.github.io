懂吃

个人私有餐厅打卡 × 算法推荐 × 好友分享推荐 × 零商业化。
一个属于自己的美食外脑 — 把每一次吃饭变成可回忆、可检索、可分享的美食记忆。

📄 查看 Markdown 介绍 📂 完整 README
CORE

三大核心模块

一款只服务于自己的餐厅管理 App。不是大众点评的替代品,而是你的私人美食外脑 — 每次吃饭都沉淀成可回忆、可检索、可分享的记忆。

🏷️
餐厅卡片库
一店一卡:位置 / 菜系 / 招牌菜 / 多次打卡记录 / 心情标签。沉淀你的美食记忆。
🤖
算法推荐
基于历史打卡 × 心情 × 时间 × 季节,主动推荐"今晚去哪吃"。5 家真实打卡即可冷启动。
👥
好友分享
私密朋友圈,只有懂吃的人之间互相推荐,不公开社交。熟人间的真心推荐。
🔒
零商业化
永远不接入商家推广 / 永远不开"商家入驻" / 永远不根据"付费排名"调整推荐。
